To pause a {props.name} using the CLI, run:
<CodeBlock>
  vcluster pause [name]
</CodeBlock>

<br></br>
:::info Automatic Resume

<span>
  Note that the {props.name} will automatically resume, once you run a
  kubectl command or send a Kubernetes request via another tool to the{" "}
  {props.name}.
</span>
:::
