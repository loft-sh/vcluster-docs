<Flow id="delete-namespace-kubectl">
  <Step>

If you do not have a kube-context for the cluster where this namespace is located in, you can set up a kube-context using:
```bash
loft use cluster [cluster-name]
```

  </Step>
  <Step>

Delete the namespace using kubectl:
```bash
kubectl delete namespace [namespace-name]

# Alternatively:
# kubectl delete -f namespace.yaml
```

  </Step>
</Flow>
