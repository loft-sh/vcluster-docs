<Flow id="list-namespaces-kubectl">
  <Step>

If you do not have a kube-context for the cluster where you want to list namespaces in, you can set up a kube-context using:
```bash
loft use cluster [cluster-name]
```

  </Step>
  <Step>

Run this kubectl command to get a list of namespaces:
```bash
kubectl get namespaces
```

  </Step>
</Flow>
