Templates can be <i>versioned</i>. Versioning a template allows you
to make changes over time to a template while retaining the current version/iteration. Resources (i.e. virtual clusters, spaces) that are deployed with templates that have updated versions will show up with a warning
indicating that a newer template version is available.

## Creating A Template Version

You can add new versions to your templates via the UI. The steps are the same for all resources. 

:::info Semantic Versioning
While not strictly mandatory, you should follow standard [semantic versioning](https://semver.org)
practices for your template versioning. vCluster Platform refers to the version numbers provided when determining
whether to indicate if a template has a newer version available.
:::

<Flow id="create-template-version">
  <Step>
    Select the <NavStep>Templates</NavStep> field on the left menu bar.
  </Step>
  <Step>Click the option for the resource you want to version on the templates pane.</Step>
  <Step>
    Find the template you would like to add a new version to in
    the list of templates. Click the small drop down button next
    to the name of your desired template. From the drop down
    menu, click the <Label>Add A New Version</Label> button.
  </Step>
  <Step>
    In the pop up box, enter the new version string for your App. Once done,
    click the
    <Button>Add New Version</Button>.
  </Step>
  <Step>
    In the drawer that appears from the right, make any modifications desired
    for your new template version.
  </Step>
  <Step>
    Click the <Button>Save Changes</Button> button.
  </Step>
</Flow>

## Updating a Template 

You can change the template and/or template version in the UI.

<Flow id="change-vcluster-template-ui">
  <Step>
    Select the <NavStep>Projects</NavStep> field on the left menu bar.
  </Step>
  <Step>
    From the project drop-down menu, select the project containing the resource you'd like to update.
  </Step>
  <Step>Click the option on the project pane.</Step>
  <Step>
    Click the dropdown arrow next to the name of the resource you'd like
    to modify.
  </Step>
  <Step>
    In the dropdown menu, select <Label>Change Template</Label>.
  </Step>
  <Step>
    In the pop-up window, select the name of the template from the drop down
    menu -- the currently applied template will already be selected, so if you
    are just changing the template version you don't need to do anything for
    this step!
  </Step>
  <Step>
    If the template you are selecting has multiple versions, you will see an
    additional version dropdown menu next to the template selector dropdown,
    select your desired template version from this menu.
  </Step>
  <Step>
    Click the <Button>Change Template</Button> button.
  </Step>
</Flow>
