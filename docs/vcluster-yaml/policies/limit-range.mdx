`limitRange`:
- Also part of [Isolated Mode](https://www.vcluster.com/docs/security/isolated-mode)

`limitRange.default`, `limitRange.defaultRequest`:
- These correspond to the default & defaultRequest properties described in the [kubernetes documentation](https://kubernetes.io/docs/concepts/policy/limit-range/)
- Similar to resource quotas, it's important to note that the `LimitRange` will be created in the same namespace as the vCluster on the host, and only apply to synced resources such as Pods

`limitRange.annotations`:
- annotations to add to the LimitRange resource

`limitRange.labels`:
- labels to add to the LimitRange resource
