`endpoints.enabled`:
- covered by the generic section here: https://www.vcluster.com/docs/syncer/config#enable-or-disable-synced-resources

Syncs all [Endpoints](https://kubernetes.io/docs/concepts/services-networking/service/) from the virtual cluster to the host cluster. This is enabled by default.

### Sync all Endpoints from virtual to host cluster
```YAML
sync:
  toHost:
    endpoints:
      enabled: true
```
