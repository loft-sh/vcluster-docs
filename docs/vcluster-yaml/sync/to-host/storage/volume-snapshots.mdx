`volumeSnapshots.enabled`:
- covered by the generic section here: https://www.vcluster.com/docs/syncer/config#enable-or-disable-synced-resources
- Added to close issue: https://github.com/loft-sh/vcluster/issues/258

Sync [VolumeSnapshot resources](https://kubernetes.io/docs/concepts/storage/volume-snapshots/) from the virtual cluster to the host cluster.

By default, VolumeSnapshot syncing is disabled, and creating a VolumeSnapshot custom resource in the virtual cluster has no effect on the host cluster.

See [Storage](/architecture/storage.mdx) for details on how this feature works.

### Sync VolumeSnapshots from virtual to host cluster
```YAML
sync:
  toHost:
    volumeSnapshots:
      enabled: true
```
