
These tables lists all resources that vCluster can sync. You can activate or deactivate via the `vcluster.yaml`. By default, certain resources are already activated. By default, vCluster runs with a minimal set of RBAC permissions to allow execution in restricted environments. Certain resources require extra permissions, which are automatically given to the vCluster ServiceAccount if you enable the resource sync.
