`ingressClasses.enabled`:
- covered by the generic section here: https://www.vcluster.com/docs/syncer/config#enable-or-disable-synced-resources
- original issue: https://github.com/loft-sh/vcluster/issues/663

Sync IngressClasses from the host cluster to the virtual cluster. This is disabled by default. However, if you enable [ingresses syncing](/vcluster/vcluster-yaml/sync/to-host/networking/ingresses), then this `ingressClasses` option is automatically be enabled. Disabling `ingressClasses` can lead to unexpected results if you have enabled `ingresses` syncing.

### Sync IngressClasses from the host cluster to the virtual cluster

```YAML
sync:
  fromHost:
    ingressClasses:
      enabled: true
```
