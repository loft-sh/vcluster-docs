Sync IngressClass resources from the host cluster to the virtual cluster. This is disabled by default. However, if you enable [ingress syncing](/vcluster/vcluster-yaml/sync/to-host/networking/ingresses), then this `ingressClasses` option is automatically be enabled. Disabling `ingressClasses` can lead to unexpected results if you have enabled `ingresses` syncing.
