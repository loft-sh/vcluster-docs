Access keys let you authenticate with vCluster Platform API endpoints and vCluster Platform CLI in non-interactive environments such as from within CI/CD pipelines.

## Example Owned Access Key

An example Owned Access Key:

```yaml
apiVersion: management.loft.sh/v1
kind: OwnedAccessKey
metadata:
  creationTimestamp: null
  name: my-access-key
spec:
  displayName: My Access Key
  ttl: 1728000
  type: User
  user: my-user
status: {}
```
