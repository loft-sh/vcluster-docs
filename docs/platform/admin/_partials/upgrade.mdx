Upgrade vCluster Platform via:

<Tabs
  defaultValue="cli"
  values={[
    { label: 'CLI', value: 'cli', },
    { label: 'helm', value: 'helm', },
  ]
}>
<TabItem value="cli">

To upgrade vCluster Platform via vCluster Platform CLI, run:

```bash
vcluster pro start --upgrade --version=$VERSION --values=vcluster-pro.yaml
```

</TabItem>
<TabItem value="helm">

To upgrade vCluster Platform via `helm`, run:

```bash
helm upgrade loft vcluster-control-plane -n vcluster-pro --repository-config '' --repo https://charts.loft.sh \
  --version $VERSION \
  --reuse-values \
  -f vcluster-pro.yaml
```

</TabItem>
</Tabs>
