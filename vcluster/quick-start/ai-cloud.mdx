This guide is in progress.

vCluster Standalone lets you deliver a managed Kubernetes platform — similar to hyperscaler offerings such as EKS — from your own bare metal or VM infrastructure, with no dependency on any external Kubernetes distribution.

Each tenant receives a dedicated cluster provisioned in seconds on the same Standalone infrastructure. No physical cluster per customer. No control plane overhead per tenant.

## What you will need

- One or more bare metal servers or VMs to run the Standalone control plane
- Linux machines to join as private worker nodes per tenant
