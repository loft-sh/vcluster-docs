#### `secretRef` <span className="config-field-required" data-required="false">required</span> <span className="config-field-type">object</span> <span className="config-field-default"></span> <span className="config-field-enum"></span> <span data-pro="false" className="config-field-pro">pro</span> {#platform-api-secretRef}

SecretRef defines where to find the platform access key and host. By default, vCluster will search in the following locations in this precedence:
* platform.api.accessKey
* environment variable called LICENSE
* secret specified under platform.api.secretRef.name
* secret called "vcluster-platform-api-key" in the vCluster namespace
