By default, this is disabled. 

Sync IngressClass resources from the host cluster to the virtual cluster. 

### Enable Syncing IngressClasses from the Host to Virtual Cluster


```YAML
sync:
  fromHost:
    ingressClasses:
      enabled: true
```
