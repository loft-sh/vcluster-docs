
### Worker Nodes

Decide where you want your worker nodes to come from:

- **[Nodes from the host cluster](../../worker-nodes/host-nodes/README.mdx)** - (Default) All worker nodes of the shared host cluster are used by the virtual cluster and all resources are synced to the single namespace that the vCluster is deployed on.
  - **[Syncing Namespaces](../../../configure/vcluster-yaml/sync/from-host/nodes.mdx)** - Resources are synced to mapped namespaces on the host cluster.
  - **[Isolated workloads](../../worker-nodes/host-nodes/isolated-workloads.mdx)** - Different options to isolate a workload in a vCluster.
- **[Private Nodes](../../worker-nodes/private-nodes/README.mdx)** - Enable adding individual nodes to the virtual cluster.
