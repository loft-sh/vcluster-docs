Deploy `nginx-ingress` controller to your cluster. Update `$PLATFORM_VERSION` with a valid vCluster Platform version.

```bash
helm upgrade --install ingress-nginx ingress-nginx \
  --repository-config='' \
  -n ingress-nginx \
  --create-namespace \
  --repo https://kubernetes.github.io/ingress-nginx \
  --set-string controller.config.hsts=false \
  --wait \
  --version $PLATFORM_VERSION
```
