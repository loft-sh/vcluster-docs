Different options may be configurable on a template, but the following is a list of the main options.

<Flow id="create-vcluster-template-optional-ui">
  <Step>
    <Label>Template Parameters</Label> expose values that users can modify when deploying a virtual cluster from this template.
  </Step>
  <Step>
    <Label>Apps</Label> select pre-defined applications to install into the virtual cluster. <Label>Objects</Label> define Kubernetes manifests that should be created in the virtual cluster.
  </Step>
  <Step>
    <Label>Management Access</Label> control who has permission to view or edit this template.
  </Step>
</Flow>
