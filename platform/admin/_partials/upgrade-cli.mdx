To upgrade vCluster Platform via vCluster Platform CLI, run:

```bash
vcluster pro start --upgrade --version=$VERSION --values=vcluster-pro.yaml
```

:::info Versions and Values
The `$VERSION` variable above is an environment variable set to your desired vCluster Platform version to
deploy. The `vcluster-pro.yaml` file is an optional YAML file that contains Helm values you would like
to use when upgrading your vCluster Platform deployment.
:::
