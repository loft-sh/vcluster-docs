## Delete project secret

To delete a project secret follow these steps:

<Flow id="secret-delete-project-secret-data">
  <Step>
    Navigate to the <NavStep>Projects</NavStep> view using the menu on the left
  </Step>
  <Step>
    Select the project you'd like to configure using the drop down menu
  </Step>
  <Step>
    Click on <NavStep>Secrets</NavStep>
  </Step>
  <Step>
    Click the <Label>Delete</Label> option in the drop-down menu next to the name of your secret
  </Step>
  <Step>
    Click on the <Label>Delete</Label> button in the modal window to confirm the
    deletion
  </Step>
</Flow>
