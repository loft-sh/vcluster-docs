This API can be used to retrieve all allowed clusters of this project.

## Project Clusters example

An example Project Clusters:
```yaml
apiVersion: management.loft.sh/v1
clusters:
- apiVersion: management.loft.sh/v1
  kind: Cluster
  metadata:
    name: my-cluster
  spec:
    config: {}
  status: {}
kind: ProjectClusters
metadata: {}

```
