Shows information about the current user. Alternatively you can also provide an alternative access key to retrieve information from.

## Example Self

An example Self:

```yaml
apiVersion: management.loft.sh/v1
kind: Self
metadata:
  creationTimestamp: null
spec: {}
status:
  accessKey: my-access-key
  accessKeyType: Login
  groups:
    - loft:admins
    - system:authenticated
    - loft:authenticated
    - loft:user:*
    - loft:user:admin
  instanceID: ""
  intercomHash: ""
  subject: admin
```
