Determine the External-IP address:

```bash
kubectl get ingress -n vcluster-pro
NAME           CLASS    HOSTS                ADDRESS          PORTS     AGE
loft-ingress   <none>   vcluster-pro.mydomain.tld    x.x.x.x          80, 443   10m
```

:::info Address
If you never get a valid IP Address in the ADDRESS column shown above, you will need to work
with your Kubernetes administrator to ensure your ingress is properly configured!
:::
