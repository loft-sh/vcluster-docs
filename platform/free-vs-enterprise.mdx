# Compare pricing tiers

vCluster Platform is available in multiple tiers: **Free**, **Dev**, **Prod**, and **Scale**. The Dev, Prod, and Scale tiers are all Enterprise tiers with increasing capabilities.

For detailed pricing, see the [vCluster Pricing](https://www.vcluster.com/pricing) page.

## Tier overview

### Free tier

The Free tier provides core platform capabilities for managing virtual clusters:

- Virtual cluster management
- Connected clusters and cluster roles
- Secrets management and apps
- Namespaces and custom branding
- Basic OIDC provider

### Dev tier (Enterprise)

The Dev tier adds capabilities for development teams:

- Sleep mode for cost optimization
- High availability and multi-region mode
- Single Sign-On (SSO) authentication
- Template versioning
- Integrations (Cert Manager, ESO, Argo, Vault)

### Prod tier (Enterprise)

The Prod tier adds production-ready features:

- External database support (PostgreSQL, MySQL, RDS)
- Audit logging
- Istio integration
- Project quotas
- Scheduled snapshots

### Scale tier (Enterprise)

The Scale tier adds enterprise-scale capabilities:

- FIPS compliance
- Centralized admission control
- Multiple SSO providers
- Database connector with EKS Pod Identity
- Hybrid scheduling
- KubeVirt integration
- Netris and Kube-vip integrations

## Activate the free tier

1. **Deploy vCluster Platform**: Follow the [installation guide](./install/quick-start-guide.mdx)
2. **Activate your instance**: Enter your email when prompted in the Platform UI
3. **Verify activation**: Complete activation via the email link

No credit card required.

## Upgrade to enterprise

1. Visit the [vCluster Pricing](https://www.vcluster.com/pricing) page
2. Contact [sales](https://www.vcluster.com/enterprise-demo) for a demo
3. Your existing instance upgrades without redeployment
