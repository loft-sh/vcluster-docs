Kubernetes (K8s) is the only supported distribution. **K0s** support was removed in vCluster `v0.26` and **K3s** support was removed in `v0.33`. If your tenant clusters still use K3s, see the [K3s to K8s migration guide](../../../manage/upgrade/distro-migration.mdx) before upgrading.
:::
