## Config notes

Use `syncSettings.disableSync` and `syncSettings.rewriteKubernetesService` together.

```yaml
experimental:
  syncSettings:
    disableSync: true
    rewriteKubernetesService: true
```

This configuration disables all sync controllers and doesn't deploy CoreDNS.
