<TabItem value="vcluster">

Create a `vcluster.yaml` file with the following lines:

```yaml
controlPlane:
  statefulSet:
    security:
      podSecurityContext:
        fsGroup: 12345
      containerSecurityContext:
        runAsUser: 12345
        runAsNonRoot: true
```

Then create the vCluster with the following command:
```
vcluster create my-vcluster -f vcluster.yaml
```

</TabItem>
<TabItem value="helm">

Update the `vcluster.yaml` file.

You will need to add the `securityContext` block as shown below:

```yaml
controlPlane:
  statefulSet:
    security:
      podSecurityContext:
        fsGroup: 12345
      containerSecurityContext:
        runAsUser: 12345
        runAsNonRoot: true
```

Then, install helm chart using `vcluster.yaml` for chart values.

</TabItem>
