By default, vCluster uses embedded [SQLite](https://www.sqlite.org/index.html) with a PersistentVolume resource to store data.

You can [turn off PersistentVolume sync](../../../../sync/to-host/storage/persistent-volumes.mdx) for testing, where losing data on restarts is acceptable. 

## Disable SQLite

```yaml
controlPlane:
  backingStore:
    database:
      embedded:
        enabled: false
```

Disable SQLite if you want to use a different database. See [External database](./external.mdx) for configuration details.

:::warning
After deploying your vCluster, changing the backing store of vCluster is not supported.
:::
