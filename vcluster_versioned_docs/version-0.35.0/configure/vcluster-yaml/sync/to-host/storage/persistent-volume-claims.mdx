By default, this is enabled.

Sync [PersistentVolumeClaim](https://kubernetes.io/docs/concepts/storage/persistent-volumes/) resources from the tenant cluster to the control plane cluster.

### Disable syncing PersistentVolumeClaims from the tenant to control plane cluster

```YAML
sync:
  toHost:
    persistentVolumeClaims:
      enabled: false
```

## Patches

Use `sync.toHost.persistentVolumeClaims.patches` to transform PersistentVolumeClaim fields while syncing to the control plane cluster. See [Patching synced resources](../../patching/README.mdx) for syntax, directionality, and examples.
